# viagens--landing-page

Uma landing page de uma agência de viagens fictícia criada com HTML, CSS e JavaScript.

## Instalação

1. Em seu terminal de escolha, clone o repositório com `git clone https://github.com/arthur-tsutiya/viagens--landing-page/`.
2. Mude o seu diretório para a pasta do projeto com `cd viagens--landing-page`.
3. Abra o arquivo `index.html` em seu navegador de escolha.
